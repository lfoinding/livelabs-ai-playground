# LiveLabs AI Developer

Estimated Time: 5 minutes

## Introduction

This repo enables LiveLabs AI Developer author workflows. It brings together reusable agentic automation skills and examples for common workshop-authoring tasks.

Instead of treating authoring as only markdown editing, this focuses on the surrounding work that authors actually do before, during, and after writing: turning source materials into workshop drafts, reviewing learner flows, capturing screenshots, adding knowledge checks, adapting workshops for new audiences, and handling other repeatable authoring tasks.

The `skills/how-to/` workshop packages those examples into a LiveLabs format so authors can review the flows, reuse the prompt patterns, and build on them over time. The current labs are only the starting point. The workshop is meant to grow as new author workflows are added.

### Objectives

- identify the main author workflows currently demonstrated in the repo
- understand how source materials can become a first-draft workshop
- find the workshop entry point under `skills/how-to/`
- understand where new workflow guides can be added over time
