# LiveLabs AI Playground

Estimated Time: 5 minutes

## Introduction

This repo holds two skill-focused areas for LiveLabs work under `skills/how-to/`. One area teaches authors how to capture workshop-ready screenshots with Codex. The other area packages quiz-generation assets for LiveLabs gamification.

### Objectives

- identify the two skill areas in the repo
- find the screenshot workshop entry point
- find the quiz-skill guide and package

## Task 1: Review The Repo Layout

1. Use `skills/how-to/screenshot-skill/` for the workshop-style guide on taking LiveLabs screenshots with Codex.

2. Use `skills/how-to/quiz-skill/` for the quiz-generation guide and packaged skill assets.

3. Use these paths as the main entry points:

    - screenshot workshop renderer: `skills/how-to/screenshot-skill/workshops/sandbox/`
    - screenshot lab markdown: `skills/how-to/screenshot-skill/livelabs-screenshots-with-codex/livelabs-screenshots-with-codex.md`
    - quiz skill guide: `skills/how-to/quiz-skill/livelabs-gamification-README.md`
    - quiz skill package: `skills/how-to/quiz-skill/livelabs-gamification.zip`
